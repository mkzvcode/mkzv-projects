# MKZV Projects

A collection of front-end site projects and design studies — landing pages, product pages, and small web apps, each living in its own folder.

## Projects

### [lamzu-site](./lamzu-site)

An Apple-style concept product page for a fictional wireless gaming mouse ("Lamzu Atlantis Mini"). Built as a design study — SVG-illustrated product visuals (no stock photography), a pinned scrollytelling "anatomy" section, animated spec counters, a light/dark theme toggle, and a filterable shop catalog with a quick-view modal.

**Stack:** vanilla HTML / CSS / JavaScript — no build step, no dependencies.

**Pages:**
- `index.html` — product landing page
- `shop.html` — catalog with category filters and quick-view

To preview locally, open `lamzu-site/index.html` directly in a browser, or serve the folder with any static file server (e.g. `python -m http.server`).
